# Concept
Mastering the distinction between 'indoor' (adjective) and 'indoors' (adverb), and applying the same logic to 'outdoor/outdoors'.

## Formulas/Patterns
- **Adjective Rule (Indoor / Outdoor)**:
  - Adjectives modify nouns. They always come **before** a noun.
  - *Pattern*: [Indoor/Outdoor] + [Noun]
- **Adverb Rule (Indoors / Outdoors)**:
  - Adverbs modify verbs. they indicate **where** an action happens. They usually come **after** the verb.
  - *Pattern*: [Verb] + [Indoors/Outdoors]

## Comparative Table
| Type | Inside | Outside | Usage |
| :--- | :--- | :--- | :--- |
| **Adjective** | **Indoor** | **Outdoor** | Before a Noun |
| **Adverb** | **Indoors** | **Outdoors** | After a Verb |

## Examples
- **Indoor vs Indoors**:
  - "The hotel features a luxury **indoor** swimming pool." (Adjective before Noun)
  - "It was raining, so we decided to play tennis **indoors**." (Adverb after Verb)
- **Outdoor vs Outdoors**:
  - "I love **outdoor** activities like hiking and camping." (Adjective)
  - "Children should spend more time playing **outdoors**." (Adverb)

## IELTS Topic Applications
- **Hobbies**: When discussing hobbies, use these to specify environment.
  - "I prefer **indoor** hobbies like reading, but my brother likes playing football **outdoors**."
- **Urban Planning (Part 3)**: 
  - "City planners should provide more **outdoor** communal spaces for residents to socialize **outdoors**."

## Common Mistakes
- **Incorrect**: "I like to stay indoor." -> **Correct**: "I like to stay **indoors**."
- **Incorrect**: "He is an indoors person." -> **Correct**: "He is an **indoor** person." (Though 'homebody' is more common).
- **Tip**: Remember the **'s'** for the **S**etting of the action (where it happens).
